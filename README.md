# 🔥 火锅计时助手 (Hotpot Timer)

一个帮你涮火锅时掌控火候的微信小程序。支持多食材并行计时、到时提醒、同桌多人实时同步。

## ✨ 核心功能

- 🍲 **多种锅底** - 清汤/番茄/麻辣/牛油/菌菇，自动适配涮煮时间
- 🥩 **50+ 食材库** - 参考海底捞推荐时间，三段式（安全/推荐/最长）
- ⏱️ **多路并行计时** - 最多同时计时 8 个食材
- 🔔 **到时提醒** - 震动 + 声音 + 屏幕提示 + 订阅消息
- 👥 **同桌多人同步** - 创建房间，朋友扫码加入，全桌共享计时看板
- ⭐ **收藏组合** - 常点食材套餐一键启动
- 📜 **历史记录** - 回看每次火锅聚餐

## 🏗️ 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 微信小程序原生 + TypeScript + MobX-miniprogram |
| 后端 | Node.js 18+ / Fastify / ws |
| 存储 | 内存 Map（房间状态） + JSON（静态食材库） |
| 通信 | HTTPS + WebSocket |

## 📁 项目结构

```
hotpot-timer/
├── miniprogram/       # 微信小程序端
│   ├── pages/         # 页面
│   ├── components/    # 组件
│   ├── stores/        # MobX 状态
│   ├── services/      # 服务层（API/WS/通知）
│   ├── utils/         # 工具函数
│   └── assets/        # 图片、音频
├── server/            # 后端服务

│   └── data/          # 食材库 JSON
└── docs/              # 文档（PRD/技术方案）
```

## 🚀 开发运行

### 后端
```bash
cd server
npm install
npm run dev    # 本地开发，端口 3000
```

### 小程序
1. 用微信开发者工具打开 `miniprogram/` 目录
2. 填入自己的 AppID（测试可用测试号）
3. 编译运行
